# Exercícios de Lógica de Programação - Portugol

Bem-vindo(a) ao meu repositório de estudos! 🚀

Este repositório contém uma série de exercícios práticos desenvolvidos durante as aulas da disciplina de Algoritmos na **Faculdade Una**. O objetivo é registrar minha evolução na construção de algoritmos, manipulação de variáveis, estruturas condicionais e cálculos matemáticos.

## 🎓 Sobre os exercícios
Estes algoritmos foram propostos como parte das atividades práticas para fortalecer minha base de lógica antes da migração para linguagens de nível profissional. Os exercícios abrangem:

* [cite_start]**Entrada e Saída de Dados:** Manipulação de leitura e exibição de informações[cite: 4, 8, 14, 20, 26].
* [cite_start]**Cálculos Matemáticos:** Aplicação de fórmulas para conversão de temperaturas, médias ponderadas, consumo de combustível e cálculos comerciais[cite: 8, 14, 20, 26].
* [cite_start]**Estruturas Condicionais:** Uso de tomadas de decisão (`se`/`senao`) para validar opções do usuário[cite: 8].

## 🚀 Como executar
1. Instale o [Portugol Studio](https://portugol-studio.uol.com.br/).
2. Clone este repositório ou baixe os arquivos `.por`.
3. Abra o arquivo desejado no programa e clique em **Executar**.
